#Json Basic Structures

{Dictionarys} and [Arrays]

You can nest how ever you like....

#ex.1

{ 
	"fruit": [ 
    		"apple",
    		"pear",
    		"peach"
	]
}

#ex.2

{
    'address': 
        { 
            'street_address': "23 Elm Drive",
            'city' : "Palo Alto",
            'state': "California",
            'zipcode': "94305"
        }
}
